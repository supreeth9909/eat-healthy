# EatHealthy - Professional Food ordering

A modern, responsive, and professional food ordering web application designed for health-conscious individuals. This project is a complete overhaul of the original demo, featuring a robust modular architecture, a premium design system with dark mode, and persistent state management.

## 🚀 Features

- **Premium UI/UX**: Built with Tailwind CSS and custom design tokens for a sleek, modern look.
- **Dark Mode Support**: Seamless theme switching with persistent user preference.
- **Modular Data Management**: Decoupled menu data and logic for better maintainability.
- **Persistent Cart**: LocalStorage-backed cart and order history that survives page refreshes.
- **AI Chat Assistant**: Integrated helper to answer menu and delivery questions.
- **Secure Checkout**: Simulated multi-step payment process with order confirmation.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **CSS3 (Vanilla)**: Custom design system and smooth transitions.
- **Tailwind CSS**: Utility-first styling.
- **JavaScript (ES6+)**: Modular and functional logic.
- **Lucide Icons**: Clean, consistent iconography.

## 📂 Project Structure

```text
.

├── style.css       # Design system and custom styles
├── main.js        # Application logic and UI controllers

```

## 🚦 Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Enjoy exploring the healthy menu!
